# 進捗バーのコントラスト改善

## 背景と目的

現在のガントチャートでは、タスクバー内の進捗を `rgba(0, 0, 0, 0.28)` の半透明黒で塗りつぶしている。バー高さが 12px と細く、かつ半透明が背景に溶け込むため、進捗がわかりにくいというフィードバックを受けた。

本変更の目的は、ダークモード・ライトモードの両方で進捗部分のコントラストを高め、一目で進捗を把握できるようにすることである。

## 採用アプローチ

**A. タスク色に応じて進捗部分を暗くする**

- タスクにカスタム色が設定されている場合は、その色を暗くした実色で進捗を塗る。
- カスタム色がない場合はテーマ別のフォールバック色を使用する。
- SVG/PowerPoint エクスポートでもすでに同様のロジックを使っており、画面表示とエクスポート表示を統一できる。

## 変更詳細

### 1. 色の定義

`src/index.css` にテーマ変数 `--progress-fill` を追加する。

```css
:root {
  --progress-fill: #1e1b4b; /* ダークモード */
}

[data-theme="light"] {
  --progress-fill: #172554; /* ライトモード */
}
```

### 2. ガントバー描画

`src/components/GanttView/GanttView.tsx` の進捗矩形描画部分を変更する。

- 現状: `<rect className={styles.progressFill} />`（CSS で `rgba(0,0,0,0.28)`）
- 変更後: 進捗矩形に `fill` 属性で動的に色を指定する。
  - `task.color` が設定されている場合: `darken(task.color, 0.72)`
  - 未設定の場合: `var(--progress-fill)`

`.progressFill` の CSS クラスは不要になるが、既存の SVG 構造を壊さないよう、fill を inline で上書きする形にする。

### 3. SVG エクスポートとの整合性

`src/utils/export.ts` ではすでに `darken(task.color ?? C.taskBar, 0.72)` を使用している。未設定時のフォールバック色 `C.taskBar` を、テーマに応じて `--progress-fill` と一致する色（ダーク: `#1e1b4b`、ライト: `#172554`）に変更する。

## 対象外

- サマリーバー（親タスク）への進捗塗りつぶし追加
- マイルストーンへの進捗表示
- タスクテーブルの進捗列（数字表示）の変更

## 成功基準

- ダークモード、ライトモードいずれでも進捗バーがはっきり見える。
- タスクにカスタム色を設定しても、進捗部分が適切なコントラストを保つ。
- SVG/PNG/PowerPoint 貼り付けでも透過やノイズが発生しない。
- 既存のテストがすべて通過する。
