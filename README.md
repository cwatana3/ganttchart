# ガントチャートアプリケーション (Gantt Chart Application)

Vite + React + TypeScript で構築された、モダンで直感的な操作が可能なガントチャートアプリケーションです。美しいデザイン（ダークモード/ライトモード対応）に加え、ドラッグ＆ドロップによる各種操作や、PowerPoint 等での二次利用を考慮した高精度なエクスポート機能を搭載しています。

---

## 🚀 主な機能

### 📅 ガントチャート操作
* **タスクのドラッグ移動・リサイズ**: タスクバーをマウスでドラッグして開始日・終了日をシフトしたり、端をドラッグして期間を伸縮させることができます。
* **SHIFT＋ドラッグによる依存関係（先行）の作成**:
  * SHIFTキーを押しながらタスクバーをドラッグして他のタスクバーへ繋ぐだけで、視覚的に依存関係（矢印ライン）を設定できます。
  * **循環依存防止機能**: 無限ループとなる接続を行おうとすると警告アラートが表示され、誤った接続を未然に防ぎます。
  * 設定された依存関係は、タスクテーブルの「先行」列に自動でインデックス番号（1始まり）として反映されます。

### 📋 タスクテーブル操作
* **複数タスクの選択と一括操作**:
  * SHIFT または CTRL / ⌘ キーを押しながら行をクリックすることで、複数のタスクを同時に選択できます。
  * 選択した複数タスクは、一括でドラッグ移動（日付シフト）したり、一括削除することができます。
* **ドラッグ＆ドロップによる並び替え・階層化**:
  * タスク行をドラッグして、位置の入れ替えや、別のタスクの「中（子タスク）」へ移動して階層化させることができます。
* **ダブルクリックによる列幅の自動調整**:
  * 各ヘッダーセル、またはその境界線をダブルクリックすると、表示されているテキスト（フォント幅、インデントピクセル、展開ボタン、マイルストーンアイコン等）を正確に計算し、見切れが発生しない最適な幅に自動調整（オートフィット）されます。

* **10列のタスクテーブル**: 行番号（#）・WBS 番号・タスク名・期間・開始日・終了日・進捗・担当者・先行・メモの10列を備え、各列の幅は自動調整／手動リサイズに対応します。
* **進捗率の表示・編集**: 各タスクに 0〜100% の進捗を設定でき、ガントバー内に塗りつぶしで可視化されます。親タスクの進捗は子の期間で加重平均して自動集計します。
* **タスクの色分け**: 右クリックメニューのスウォッチからタスクバーの色を個別に変更できます。
* **メモ**: 各タスクに自由記述のメモを付与できます。

### 🔗 依存関係とスケジューリング
* **依存タイプとラグ**: FS / SS / FF / SF の4種類の依存タイプとラグ（稼働日）に対応します。「先行」列に `3` / `3SS` / `3FS+2` / `3-1` のように入力できます。
* **依存矢印のクリック削除と制約違反の警告**: ガント上の依存矢印をクリックすると確認のうえ削除できます。依存制約に違反している矢印は赤い破線で警告表示されます。
* **自動スケジューリング**: 「🔗 自動配置」をオンにすると、依存制約を満たすように後続タスクを稼働日単位で自動的に後送りします（前倒しはしません）。
* **クリティカルパス表示**: 「🛤 クリティカルパス」をオンにすると、プロジェクト完了日を左右するタスク（フロート 0）を赤枠で強調表示します。
* **ベースライン（計画）比較**: 任意の時点の計画を基準線として記録し、各タスクバーの下に計画線として表示して実績との差を可視化します。

### 🔄 連動と自動化
* **親タスクの期間・範囲の自動計算**:
  * 子タスクの期間や日付が変更されると、それらを包含するように親タスクの開始日・終了日・期間・進捗がボトムアップでリアルタイム自動再計算されます。
* **UNDO / REDO**: Ctrl+Z で元に戻し、Ctrl+Y（または Ctrl+Shift+Z）でやり直しができます。
* **キーボード操作**: Enter（直下にタスク追加）、Tab / Shift+Tab（インデント / アウトデント）、↑ / ↓（選択移動）、Ctrl+C / X / V（コピー / 切り取り / 貼り付け）、Delete（削除）に対応します。
* **タスク検索・フィルタ**: 名前・担当者・メモを横断検索し、一致タスクとその祖先のみを両ビューに表示します。
* **日本の祝日の一括追加**: カレンダー設定から年を選んで国民の祝日（振替休日・国民の休日を含む）をまとめて休日に追加できます。
* **ローカル永続化と自動バックアップ**:
  * データの更新は自動的に IndexedDB に保存されます。
  * 10分間隔・最大20世代のスナップショットを自動保存し、「🕑 履歴」から任意の時点に復元できます。

### 🗂️ 複数プロジェクト管理
* 複数のプロジェクトを切り替え・新規作成・複製・削除できます。各プロジェクトは IndexedDB に個別に保存されます（旧バージョンの単一データは自動移行されます）。

### 📤 エクスポート機能
* **PowerPoint (PPTX) 親和性の高い「透過ゼロ」SVGエクスポート**:
  * ガントチャート図を SVG ファイルとしてエクスポートできます。
  * **PPTX 図形変換への最適化**: エクスポートされる SVG には、透過カラー（`rgba`）や半透明フィルター（ドロップシャドウ等）を一切使用していません。すべて不透明なソリッドカラー（16進数）にカラーブレンドした図形要素だけで出力されるため、**PowerPoint に貼り付けて「図形に変換」した際に、シマシマのノイズが発生したり透過がおかしくなることなく、完全に編集可能な PowerPoint オートシェイプ図形に変換可能**です。
* **PNG エクスポートと印刷**: 同じ図を2倍解像度の PNG として書き出したり、ブラウザの印刷ダイアログから印刷できます。
* **エクスポート範囲の指定**: 「📤 エクスポート…」ダイアログから形式（SVG / PNG / 印刷）と期間（全体 / 範囲指定）を選べます。
* **CSV エクスポート / インポート**: BOM 付き UTF-8（Excel 互換）でタスクを入出力します。WBS 階層と「先行」列の依存（タイプ・ラグ含む）を round-trip で復元します。

## 🛠️ セットアップと開発方法

### 1. 依存関係のインストール
プロジェクトのディレクトリで以下を実行します：
```bash
npm install
```

### 2. 開発サーバーの起動
ローカル開発用サーバーを起動します（通常は `http://localhost:5173/` で起動します）：
```bash
npm run dev
```

### 3. 本番用ビルド
プロダクションビルドを実行し、`dist/` ディレクトリに最適化された静的ファイルを生成します：
```bash
npm run build
```

### 4. 単体テストの実行
Vitest を使用してテストを実行します（タスクツリーの計算、カレンダーや日付の計算等のロジックをテストします）：
```bash
npm run test
```
